## ğŸ¨ Fitur Teknis

### QR Code Generation
- Menggunakan library `qrcodegen` untuk generate QR code
- Error Correction Level: MEDIUM
- Skala: 10px per modul
- Border: 4px

### Canvas Rendering
- QR code di-render menggunakan HTML5 Canvas API
- Support untuk header custom di atas QR code
- Warna gelap: #000 (hitam)
- Warna terang: #FFF (putih)

### Naming Convention
- File yang diunduh bernama `qrdownload.png`
- Jika ada header, format: `qrdownload-[header].png`

### SHA256 Hashing
- Input otomatis di-hash menggunakan SHA256 (untuk logging/debug)
